BatteWitch

View Project

With a heartful of persistence and communication the creation of BattleWitch was fulfilled. Using HTML, CSS and JavaScript, this project focus was to create a board game into a web version with our own twist. Visual Studio Code is the main software I used to code this project, alongside Github. With BattleWitch, my partner and I were able to learn the use of JavaScript functions and further develop our problem solving skills. Challenges were not new to us since we faced a variety of them on our path. Although we had issues ranging from simple misspellings on our code to bugs and crashes, we were able to work through them all to complete a functioning BattleWitch.
